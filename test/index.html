Each test file assigns a deterministic result to window.__result. The runner executes the original and the minified file in isolated iframes and compares the two results — if they match, terser preserved the program's behaviour. Generate the .min.js files first by running Minify Current JavaScript File on each source file (or saving with minifyjs.minifyOnSave enabled).
